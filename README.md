========== Controle de Leitura - Flutter ==========

Descrição:

Este aplicativo foi desenvolvido em Flutter com o objetivo de simular um sistema simples de controle de leitura, permitindo o cadastro de usuário, login e visualização de uma lista de livros lidos.
O projeto não utiliza banco de dados ou backend, mantendo os dados apenas em memória para fins educacionais.

Objetivo:

Aplicar conceitos fundamentais do desenvolvimento mobile com Flutter, incluindo:
+ Estruturação de projeto
+ Navegação entre telas
+ Gerenciamento básico de estado
+ Criação de interfaces com widgets
  
Funcionalidades:
+ Cadastro de Usuário
Entrada de nome, email e senha
Validação de campos obrigatórios
Redirecionamento automático para login após cadastro

+ Login
Autenticação com email e senha cadastrados
Validação dos dados inseridos
Navegação para a tela principal em caso de sucesso

+ Tela Home
Mensagem de boas-vindas personalizada com o nome do usuário
Exibição de lista de livros lidos (título e autor)
Implementação utilizando ListView
Bloqueio de retorno à tela de login após autenticação

Navegação:

O aplicativo possui o seguinte fluxo:

Tela de Login -->

Tela de Cadastro -->

Retorno ao Login -->

Acesso à Home.

Navegação bloqueada para voltar ao login após entrar!

#==========================================================

Tecnologias Utilizadas:
+ Flutter
